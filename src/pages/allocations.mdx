# Allocaton

Master Allocaton

## The allocation model

The allocation model contains all the information about your allocations.

### Properties

<Properties>
  <Property name="_id" type="string">
    Unique identifier for the allocation.
  </Property>
  <Property name="name" type="string">
    The name for the allocation.
  </Property>
  <Property name="permissions" type="array<string>">
    The array of allowed permission name for the allocation.
  </Property>
  <Property name="createdBy" type="string">
    User identifier who create allocation.
  </Property>
  <Property name="createdBy" type="string">
    User identifier who update allocation.
  </Property>
  <Property name="createdAt" type="timestamp">
    Timestamp of when the allocation was created.
  </Property>
  <Property name="updatedAt" type="timestamp">
    Timestamp of when the allocation was updated.
  </Property>
</Properties>

## List all allocations {{ tag: 'GET', label: '/v1/allocations' }}

<Row>
  <Col>

    This endpoint allows you to retrieve a paginated list of all your allocations. By default, a maximum of ten allocations are shown per page.

    ### Optional attributes

    <Properties>
      <Property name="page" type="integer">
        The page number of allocations returned.
      </Property>
      <Property name="pageSize" type="integer">
        The page size number of allocations returned.
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    <CodeGroup title="Request" tag="GET" label="/v1/allocations">
    ```typescript {{ title: 'axios' }}
    const response = axios.get('/v1/allocations', {
      params: {
        filter: {
          page:1,
          pageSize: 10
        }
      }
    })
    ```
    </CodeGroup>
    <CodeGroup title="Response">
