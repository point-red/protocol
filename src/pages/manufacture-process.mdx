# Manufacture - Process

Master Manufacture - Process

## The process process

The process process contains all the information about your processs.

### Properties

<Properties>
  <Property name="_id" type="string">
    Unique identifier for the process.
  </Property>
  <Property name="name" type="string">
    The name for the process.
  </Property>
  <Property name="permissions" type="array<string>">
    The array of allowed permission name for the process.
  </Property>
  <Property name="createdBy" type="string">
    User identifier who create process.
  </Property>
  <Property name="createdBy" type="string">
    User identifier who update process.
  </Property>
  <Property name="createdAt" type="timestamp">
    Timestamp of when the process was created.
  </Property>
  <Property name="updatedAt" type="timestamp">
    Timestamp of when the process was updated.
  </Property>
</Properties>

## List all processs {{ tag: 'GET', label: '/v1/processs' }}

<Row>
  <Col>

    This endpoint allows you to retrieve a paginated list of all your processs. By default, a maximum of ten processs are shown per page.

    ### Optional attributes

    <Properties>
      <Property name="page" type="integer">
        The page number of processs returned.
      </Property>
      <Property name="pageSize" type="integer">
        The page size number of processs returned.
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    <CodeGroup title="Request" tag="GET" label="/v1/processs">
    ```typescript {{ title: 'axios' }}
    const response = axios.get('/v1/processs', {
      params: {
        filter: {
          page:1,
          pageSize: 10
        }
      }
    })
    ```
    </CodeGroup>
    <CodeGroup title="Response">
