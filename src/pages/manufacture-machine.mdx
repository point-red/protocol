# Manufacture - Machine

Master Manufacture - Machine

## The machine model

The machine model contains all the information about your machines.

### Properties

<Properties>
  <Property name="_id" type="string">
    Unique identifier for the machine.
  </Property>
  <Property name="name" type="string">
    The name for the machine.
  </Property>
  <Property name="permissions" type="array<string>">
    The array of allowed permission name for the machine.
  </Property>
  <Property name="createdBy" type="string">
    User identifier who create machine.
  </Property>
  <Property name="createdBy" type="string">
    User identifier who update machine.
  </Property>
  <Property name="createdAt" type="timestamp">
    Timestamp of when the machine was created.
  </Property>
  <Property name="updatedAt" type="timestamp">
    Timestamp of when the machine was updated.
  </Property>
</Properties>

## List all machines {{ tag: 'GET', label: '/v1/machines' }}

<Row>
  <Col>

    This endpoint allows you to retrieve a paginated list of all your machines. By default, a maximum of ten machines are shown per page.

    ### Optional attributes

    <Properties>
      <Property name="page" type="integer">
        The page number of machines returned.
      </Property>
      <Property name="pageSize" type="integer">
        The page size number of machines returned.
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    <CodeGroup title="Request" tag="GET" label="/v1/machines">
    ```typescript {{ title: 'axios' }}
    const response = axios.get('/v1/machines', {
      params: {
        filter: {
          page:1,
          pageSize: 10
        }
      }
    })
    ```
    </CodeGroup>
