# Branch

Master Branch

## The branch model

The branch model contains all the information about your branchs.

### Properties

<Properties>
  <Property name="_id" type="string">
    Unique identifier for the branch.
  </Property>
  <Property name="name" type="string">
    The name for the branch.
  </Property>
  <Property name="permissions" type="array<string>">
    The array of allowed permission name for the branch.
  </Property>
  <Property name="createdBy" type="string">
    User identifier who create branch.
  </Property>
  <Property name="createdBy" type="string">
    User identifier who update branch.
  </Property>
  <Property name="createdAt" type="timestamp">
    Timestamp of when the branch was created.
  </Property>
  <Property name="updatedAt" type="timestamp">
    Timestamp of when the branch was updated.
  </Property>
</Properties>

## List all branchs {{ tag: 'GET', label: '/v1/branchs' }}

<Row>
  <Col>

    This endpoint allows you to retrieve a paginated list of all your branchs. By default, a maximum of ten branchs are shown per page.

    ### Optional attributes

    <Properties>
      <Property name="page" type="integer">
        The page number of branchs returned.
      </Property>
      <Property name="pageSize" type="integer">
        The page size number of branchs returned.
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    <CodeGroup title="Request" tag="GET" label="/v1/branchs">
    ```typescript {{ title: 'axios' }}
    const response = axios.get('/v1/branchs', {
      params: {
        filter: {
          page:1,
          pageSize: 10
        }
      }
    })
    ```
    </CodeGroup>
    <CodeGroup title="Response">
