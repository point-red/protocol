# Warehouse

Master Warehouse

## The warehouse model

The warehouse model contains all the information about your warehouses.

### Properties

<Properties>
  <Property name="_id" type="string">
    Unique identifier for the warehouse.
  </Property>
  <Property name="name" type="string">
    The name for the warehouse.
  </Property>
  <Property name="permissions" type="array<string>">
    The array of allowed permission name for the warehouse.
  </Property>
  <Property name="createdBy" type="string">
    User identifier who create warehouse.
  </Property>
  <Property name="createdBy" type="string">
    User identifier who update warehouse.
  </Property>
  <Property name="createdAt" type="timestamp">
    Timestamp of when the warehouse was created.
  </Property>
  <Property name="updatedAt" type="timestamp">
    Timestamp of when the warehouse was updated.
  </Property>
</Properties>

## List all warehouses {{ tag: 'GET', label: '/v1/warehouses' }}

<Row>
  <Col>

    This endpoint allows you to retrieve a paginated list of all your warehouses. By default, a maximum of ten warehouses are shown per page.

    ### Optional attributes

    <Properties>
      <Property name="page" type="integer">
        The page number of warehouses returned.
      </Property>
      <Property name="pageSize" type="integer">
        The page size number of warehouses returned.
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    <CodeGroup title="Request" tag="GET" label="/v1/warehouses">
    ```typescript {{ title: 'axios' }}
    const response = axios.get('/v1/warehouses', {
      params: {
        filter: {
          page:1,
          pageSize: 10
        }
      }
    })
    ```
    </CodeGroup>
    <CodeGroup title="Response">
