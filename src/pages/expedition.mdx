# Expedition

Master Expedition

## The expedition model

The expedition model contains all the information about your expeditions.

### Properties

<Properties>
  <Property name="_id" type="string">
    Unique identifier for the expedition.
  </Property>
  <Property name="name" type="string">
    The name for the expedition.
  </Property>
  <Property name="permissions" type="array<string>">
    The array of allowed permission name for the expedition.
  </Property>
  <Property name="createdBy" type="string">
    User identifier who create expedition.
  </Property>
  <Property name="createdBy" type="string">
    User identifier who update expedition.
  </Property>
  <Property name="createdAt" type="timestamp">
    Timestamp of when the expedition was created.
  </Property>
  <Property name="updatedAt" type="timestamp">
    Timestamp of when the expedition was updated.
  </Property>
</Properties>

## List all expeditions {{ tag: 'GET', label: '/v1/expeditions' }}

<Row>
  <Col>

    This endpoint allows you to retrieve a paginated list of all your expeditions. By default, a maximum of ten expeditions are shown per page.

    ### Optional attributes

    <Properties>
      <Property name="page" type="integer">
        The page number of expeditions returned.
      </Property>
      <Property name="pageSize" type="integer">
        The page size number of expeditions returned.
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    <CodeGroup title="Request" tag="GET" label="/v1/expeditions">
    ```typescript {{ title: 'axios' }}
    const response = axios.get('/v1/expeditions', {
      params: {
        filter: {
          page:1,
          pageSize: 10
        }
      }
    })
    ```
    </CodeGroup>
    <CodeGroup title="Response">
