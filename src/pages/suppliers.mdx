# Supplier

Master Supplier

## The supplier model

The supplier model contains all the information about your suppliers.

### Properties

<Properties>
  <Property name="_id" type="string">
    Unique identifier for the supplier.
  </Property>
  <Property name="name" type="string">
    The name for the supplier.
  </Property>
  <Property name="permissions" type="array<string>">
    The array of allowed permission name for the supplier.
  </Property>
  <Property name="createdBy" type="string">
    User identifier who create supplier.
  </Property>
  <Property name="createdBy" type="string">
    User identifier who update supplier.
  </Property>
  <Property name="createdAt" type="timestamp">
    Timestamp of when the supplier was created.
  </Property>
  <Property name="updatedAt" type="timestamp">
    Timestamp of when the supplier was updated.
  </Property>
</Properties>

## List all suppliers {{ tag: 'GET', label: '/v1/suppliers' }}

<Row>
  <Col>

    This endpoint allows you to retrieve a paginated list of all your suppliers. By default, a maximum of ten suppliers are shown per page.

    ### Optional attributes

    <Properties>
      <Property name="page" type="integer">
        The page number of suppliers returned.
      </Property>
      <Property name="pageSize" type="integer">
        The page size number of suppliers returned.
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    <CodeGroup title="Request" tag="GET" label="/v1/suppliers">
    ```typescript {{ title: 'axios' }}
    const response = axios.get('/v1/suppliers', {
      params: {
        filter: {
          page:1,
          pageSize: 10
        }
      }
    })
    ```
    </CodeGroup>
