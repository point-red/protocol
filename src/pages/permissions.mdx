# Permissions

Master Permissions

## The permission model

The permission model contains all the information about your roles.

### Properties

<Properties>
  <Property name="_id" type="string">
    Unique identifier for the permission.
  </Property>
  <Property name="group" type="string">
    The group name for the permission. It used for grouping permission on the web
  </Property>
  <Property name="name" type="string">
    The name for the permission.
  </Property>
</Properties>

---

## List all permissions {{ tag: 'GET', label: '/v1/permissions' }}

<Row>
  <Col>

    This endpoint allows you to retrieve a paginated list of all your roles. By default, a maximum of ten roles are shown per page.

    ### Optional query

    <Properties>
      <Property name="group" type="string">
        The group name for the permission. It used for grouping permission on the web
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    <CodeGroup title="Request" tag="GET" label="/v1/permissions">

    const response = axios.get('/v1/permissions', {
      params: {
        filter: {
          group: 'users',
        }
      }
    })
