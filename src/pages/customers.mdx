# Customer

Master Customer

## The customer model

The customer model contains all the information about your customers.

### Properties

<Properties>
  <Property name="_id" type="string">
    Unique identifier for the customer.
  </Property>
  <Property name="name" type="string">
    The name for the customer.
  </Property>
  <Property name="permissions" type="array<string>">
    The array of allowed permission name for the customer.
  </Property>
  <Property name="createdBy" type="string">
    User identifier who create customer.
  </Property>
  <Property name="createdBy" type="string">
    User identifier who update customer.
  </Property>
  <Property name="createdAt" type="timestamp">
    Timestamp of when the customer was created.
  </Property>
  <Property name="updatedAt" type="timestamp">
    Timestamp of when the customer was updated.
  </Property>
</Properties>

## List all customers {{ tag: 'GET', label: '/v1/customers' }}

<Row>
  <Col>

    This endpoint allows you to retrieve a paginated list of all your customers. By default, a maximum of ten customers are shown per page.

    ### Optional attributes

    <Properties>
      <Property name="page" type="integer">
        The page number of customers returned.
      </Property>
      <Property name="pageSize" type="integer">
        The page size number of customers returned.
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    <CodeGroup title="Request" tag="GET" label="/v1/customers">
    ```typescript {{ title: 'axios' }}
    const response = axios.get('/v1/customers', {
      params: {
        filter: {
          page:1,
          pageSize: 10
        }
      }
    })
    ```
