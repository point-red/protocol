# Users

Master Users

## The user model

The user model contains all the information about your users.

### Properties

<Properties>
  <Property name="_id" type="string">
    Unique identifier for the user.
  </Property>
  <Property name="name" type="string">
    The name for the user.
  </Property>
  <Property name="permissions" type="array<string>">
    The array of allowed permission name for the user.
  </Property>
  <Property name="createdBy" type="string">
    User identifier who create user.
  </Property>
  <Property name="createdBy" type="string">
    User identifier who update user.
  </Property>
  <Property name="createdAt" type="timestamp">
    Timestamp of when the user was created.
  </Property>
  <Property name="updatedAt" type="timestamp">
    Timestamp of when the user was updated.
  </Property>
</Properties>

## List all users {{ tag: 'GET', label: '/v1/users' }}

<Row>
  <Col>

    This endpoint allows you to retrieve a paginated list of all your users. By default, a maximum of ten users are shown per page.

    ### Optional attributes

    <Properties>
      <Property name="page" type="integer">
        The page number of users returned.
      </Property>
      <Property name="pageSize" type="integer">
        The page size number of users returned.
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    <CodeGroup title="Request" tag="GET" label="/v1/users">
    ```typescript {{ title: 'axios' }}
    const response = axios.get('/v1/users', {
      params: {
        filter: {
          page:1,
          pageSize: 10
        }
      }
    })
    ```
    </CodeGroup>

## Assign permissions {{ tag: 'PATCH', label: '/v1/users/:id/assign-permissions' }}

<Row>
  <Col>

    This endpoint allows you to perform an update on a user to assign permission.

    ### Optional attributes

    <Properties>
      <Property name="permissions" type="array<string>">
        The array of permission name for the user.
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    <CodeGroup title="Request" tag="PATCH" label="/v1/users/637d83d15d2be122007524bf">
    ```typescript {{ title: 'axios' }}
    const response = axios.patch('/v1/users/637d83d15d2be122007524bf', {
      permissions: [
        {
          name: 'create-user',
          assign: true
        },
        {
          name: 'read-user',
          assign: false
        }
      ]
    })
    ```
    </CodeGroup>
    <CodeGroup title="Response">
    ```json {{ title: '204' }}
    // empty response
    ```
    </CodeGroup>
  </Col>
</Row>
